Задача состоит в вычислении по полиномам их вектора значений. Данные на вход поступают одновременно из нескольких файлов (или потоков), обработка распределяется на несколько процессов.



Программа принимает на вход 3 параметра: число переменных, число файлов(readers) и число процессов, отвечающих за обработку данных (solvers).

Входные данные находятся в файлах вида input_i.txt, где i начнается с 1 и заканчивается числом файлов.
Выходные данные соответственно будут находиться в файлах вида output_i.txt.

Примеры входных файлов (и полиномов в них) прилагаются.
