1.(tree.py) Реализовано решающее дерево на основе ID3. Основное отличие - использованные аттрибуты не отбрасываются. Формулы для энтропии и информации также можно посмотреть в описании данного алгоритма. В качестве параметра для дерева выступает максимальная глубина. 



2.(boosting.py) Реализован бустинг для написанных деревьев. Параметры включают в себя количество деревьев и их максимальную глубину.


3.(test.py) Файл содержит функции для генерации случайных данных для двумерного случая с возможностью добавления шума. Также присутствуют средства для визуализации работы алгоритма с примером использования. Прилагаются картинки.
